# Tech E-Shop

A dynamic e-commerce web application developed using:

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features:

- User Registration
- User Login / Logout
- Product Categories
- Product Filtering
- Shopping Cart
- Session Management
- Checkout Simulation
- Password Strength Indicator
- Password Visibility Toggle

## Database:

Tables:
- customers
- categories
- products

## Installation:

1. Clone repository
2. Import database into MySQL
3. Configure db.php
4. Run Apache & MySQL using XAMPP
5. Open localhost/Tech-Eshop
